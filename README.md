# Installation

⚠️  This process will wipe the disk labeled nvme0n1

1. Boot into [NixOS](https://nixos.org/download/) installation media

2. Clone repository

```
$ git clone https://github.com/leeghimire/dotfiles
$ cd dotfiles
```

3. Run install script

```
# ./install.sh
```
